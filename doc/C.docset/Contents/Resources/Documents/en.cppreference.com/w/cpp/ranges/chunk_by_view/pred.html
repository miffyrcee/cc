[edit]
 
constexpr const Pred& pred() const;
 (since C++23)

Returns a reference to the contained Pred object. Equivalent to return *pred_;.

The behavior is undefined if pred_ does not contain a value (that is, pred_.has_value() is false).

[edit] Parameters

(none)

[edit] Return value

A copy of the underlying view.
